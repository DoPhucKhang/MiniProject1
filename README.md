# Dự án Mini-project 1: Trang HTML chuẩn đầu tiên

## Phân vùng cấu trúc theo ảnh thiết kế UI:
- **header**: Hiển thị tiêu đề chính hệ thống học tập.
- **nav**: Chứa danh sách menu liên kết nhanh (ul, li).
- **main**: Sử dụng `.section-container` phân chia layout 2 cột.
  - *Cột 1*: Chứa thẻ ảnh minh họa `assets/ui-sketch.png` kèm alt text.
  - *Cột 2*: Chứa bảng tổng hợp dữ liệu `table` và `form` thu thập thông tin có đầy đủ `label/name`.
- **footer**: Chân trang lưu thông tin bản quyền của Đỗ Phúc Khang.
